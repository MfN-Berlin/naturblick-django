3. Logging of visits to the Internet pages and access to data and web services
When accessing the MfN's internet offers, access data relevant to data protection and data security are stored in a log file. Google Analytics is not used for this purpose. Depending on the access protocol used, the protocol data record contains information with the following contents:

  • IP address of the requesting computer
  • Host name of the IP address
  • Name of the requested web page, file or action
  • Date and time of the request
  • Amount of data transmitted
  • Access status of the web server (file transferred, file not found, command not executed, etc.)
  • Description of the web browser used
  • URL from which the access was initiated.


The stored data is used exclusively for the purpose of identification and tracing of unauthorised access attempts or access to the web server as well as for general statistical evaluations. Profiles of individual users are not created. The evaluation is carried out by authorised employees of the Museum für Naturkunde Berlin. The logged data is stored for up to three months, and all non-anonymised data is then deleted. The data in the log file are stored separately from all personal data provided by a person concerned.

3. Use and disclosure of personal data

Log data:

When using the Naturblick app, usage data relevant to data protection and data security is stored in a log file. Depending on the access protocol used, the log dataset may include information with the following contents:

  • Audio and image recordings
  • Observation metadata (species name, geocoordinates, time, quantity, notes)
  • Crash reports (stack traces, type of crash, trends and version of the device and software)
  • Device ID and metadata of identification results (coordinates, time, identification history)
  • Anonymous usage data (information about your interactions with the app)
  • Event data (details of specific user actions such as clicking on buttons)
  • Device data (information about your device, including device type, operating system and screen resolution).


The stored data is collected and used exclusively for scientific purposes. Once the data is no longer needed, it will be deleted. The scope of the data collected may also result from technical requirements. No profiles of individual users are created. The analysis is carried out by authorized staff of the Museum für Naturkunde Berlin. Usage data is stored separately from all personal data provided by an affected person. There is no unauthorized disclosure of data to unrelated third parties. Disclosure to government authorities occurs only within the framework of mandatory legal provisions or in cases of attacks on our systems for the purpose of criminal prosecution.

Real names:

Users are free to provide a name for the authorship of media and observations. The use of real names is not excluded and is therefore possible on a voluntary basis. Observations (species name, geocoordinates, time, quantity) and media are licensed under CC BY-SA 4.0 in accordance with the terms of use. According to this license, observation data and media are shared and published only in connection with the provided author name. Observations and media are passed on to scientific projects and, after a data protection and content review, published on the Naturblick map.

E-mail address:

To register an account, we require your e-mail address. Registration only becomes valid after you have clicked on a confirmation link sent to you by us. The e-mail address will be used exclusively for account management. Registration/login is voluntary and can be revoked at any time. Your personal data (e-mail address) will be deleted from our system when the account is deleted. Data processing is carried out in compliance with applicable data protection regulations.
